## 🚀 Vamos começar

Clone esse projeto e acesse os arquivos.

```bash
git clone https://github.com/thiagomarim/dogs-next.git

cd dogs-next

```

Execute esse comando para instalar as dependências.

```bash
npm install

npm start
```
